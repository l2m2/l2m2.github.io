re

Python中有一个正则相关的模块，re.

你只需要引入该模块即可。

re.findall()

re.split()

re.sub()

re.sub(pattern, replace, string, count=0, flags=0)

返回用replace替换pattern匹配的字符串。

如果没有匹配，将返回原字符串

只替换前2个

使用re.subn()会返回替换后的字符串和替换的个数。

re.search()

re.search(pattern, string, flags=0)

扫描字符串以查找正则匹配的第一个位置，然后返回相应的匹配对象。 如果没有匹配返回None。
