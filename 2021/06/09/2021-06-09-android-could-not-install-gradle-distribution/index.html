解决Could not install Gradle distribution from ...的错误

Jun 9, 2021 | Java | 阅读

在Windows 10 上安装完Android Studio后, 按照官方指南创建工程后，出现如下错误：

解决方案：

  1. 根据提示手动下载错误信息中的https://services.gradle.org/distributions/gradle-6.7.1-bin.zip

  2. 进入%USERPROFILE%/.gradle/wrapper/dists/gradle-6.7.1-bin/bwlcbys1h7rz3272sye1xwiv6目录下。随机字符串bwlcbys1h7rz3272sye1xwiv可能有差异。

    删除该目录下原来的所有文件。

  3. 将步骤1中下载好的zip包拷贝到%USERPROFILE%/.gradle/wrapper/dists/gradle-6.7.1-bin/bwlcbys1h7rz3272sye1xwiv6下。

  4. 重启Android Studio。
