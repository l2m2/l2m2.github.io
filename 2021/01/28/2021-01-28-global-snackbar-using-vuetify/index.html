给Vue项目添加全局的消息条

Jan 28, 2021 | 阅读

本文演示一下如何为Vue/Vuetify项目添加全局的消息条（也叫toasts, snackbar）.

我们期待的用法大概长这样

Store

把数据存在store中

store/snackbar.js

并在store/index.js中引入它

Plugin

为了在全局都可以使用，我们需要创建一个插件

plugins/snackbar.js

需要在注册到Vue的全局对象上

main.js

Component

消息条的组件则直接用Vuetify的就好了。

components/Snackbar.vue

Layout

把组件添加到应用中

App.vue

Usage

可以在任意页面中使用啦。

在axios的response钩子中提示错误：

utils/request.js
